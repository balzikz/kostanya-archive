# Kostanya Archive

Стартовая версия сайта цифрового архива клана Костания.

## Файлы

- `index.html` — главная страница
- `style.css` — оформление и адаптивность
- `script.js` — анимации и мобильное меню
- `assets/kostanya-flag.jpg` — основной флаг

## Как открыть

Просто открой `index.html` в браузере.

## Как опубликовать через GitHub Pages

1. Создай репозиторий `kostanya-archive`.
2. Загрузи все файлы из этой папки в корень репозитория.
3. Открой `Settings → Pages`.
4. Выбери `Deploy from a branch`.
5. Укажи ветку `main` и папку `/root`.
6. Сохрани настройки.

После публикации сайт будет доступен по адресу:

`https://ТВОЙ-НИК.github.io/kostanya-archive/`
