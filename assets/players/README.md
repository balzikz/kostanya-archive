# Скины игроков

Сюда загружаются PNG-файлы скинов игроков.

Рекомендуемый формат:

- полный Minecraft-скин в PNG;
- размер 64×64;
- прозрачный фон;
- названия латиницей, маленькими буквами, без пробелов.

Примеры:

- `daryn.png`
- `balzikz.png`
- `player_name.png`

Не использовать JPG: он портит прозрачность и пиксельные границы.
