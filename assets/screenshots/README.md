# Совместные скриншоты Костании

В эту папку загружаются совместные скриншоты, памятные кадры, изображения баз, событий и важных эпизодов истории клана.

## Рекомендуемый формат

- PNG или JPG;
- хорошее исходное качество;
- без лишнего сжатия;
- понятные названия латиницей, маленькими буквами и без пробелов.

## Примеры названий

- `2023-foundation-group.png`
- `2024-main-base.jpg`
- `2025-clan-meeting.png`
- `2026-anniversary.png`

При большом количестве файлов их можно будет разделить по годам:

- `assets/screenshots/2023/`
- `assets/screenshots/2024/`
- `assets/screenshots/2025/`
- `assets/screenshots/2026/`
